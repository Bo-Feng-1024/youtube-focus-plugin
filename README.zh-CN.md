# YouTube Focus

[English](README.md)

一个轻量的 Chrome 扩展，把 YouTube 顶栏里让人分心的元素藏起来，按一下快捷键又能恢复。
比 Unhook 更克制——只动你指定的那几个元素，别的都不碰。

> 本文件是英文 `README.md` 的中文版。

隐藏**仅在观看页 `youtube.com/watch` 生效**;浏览页（首页、搜索、频道、订阅……）完全不处理。

## 隐藏

顶栏：

- 左上角菜单（三道杠）按钮
- 左上角 YouTube Logo
- 顶部搜索框（含语音搜索按钮）
- Create 按钮
- 通知铃铛
- 用户头像
- PocketTube 扩展注入的 **Tags** 按钮

视频下方：

- 频道信息块（头像、频道名、订阅数）
- 订阅按钮
- PocketTube 扩展注入的 **Add to group** 按钮
- 整排操作按钮（赞、踩、分享、Ask、保存、更多）
- 右侧推荐/相关视频（含直播聊天）
- 评论区

视频简介**默认收起**;标题下方会注入一个「Show description」小按钮,点一下即可
展开全文。展开/收起的状态会被记住(跨视频、跨标签页),和专注模式开关一样。

## 保留

视频标题(简介一键即可展开）。

## 使用

- 默认快捷键 **Alt+Shift+F**（Mac 上 Alt = Option），或点击工具栏扩展图标，即可切换。
- 状态会被记住：跨标签页、重开浏览器后都保持上次的开/关。

## 安装（开发者模式，无需构建）

1. 打开 `chrome://extensions`，右上角开启「开发者模式」。
2. 点「加载已解压的扩展程序」，选择本文件夹。
3. 打开 YouTube，按 `Alt+Shift+F` 试试。
4. 想改快捷键：`chrome://extensions/shortcuts`。

## 改完代码后

在扩展卡片上点 ↻ 重新加载，再刷新 YouTube 标签页。

## 某个元素没藏住？

多半是 YouTube 改了页面结构。优先去 `src/hide.css` 调整选择器；
若是 PocketTube 的 Tags 按钮没生效，帮忙在页面上右键检查那个按钮、
把它的 class 发给我，我换成更精确的选择器。
